# NHFAE 分阶段创新与自训路线图

## 目标
将现有 Phase D.3 工程原型升级为原生一体化 NHFAE 架构，逐步完成：
1. 数学同构验证（离散身份 + 连续几何）
2. 结构同构实现（共享主干 + 双头输出）
3. 训练同构闭环（自训编解码器 + 端到端目标）

## 当前状态（已完成）
1. Phase-First Flow 已验证高 SNR 透明性恢复趋势。
2. 高频 SDR 专项评估工具已具备。
3. 现有流程可作为 NHFAE 的可用基线（Baseline-0）。

## Phase E0: 数学同构最小化验证
### 目标
在不重写主干的前提下，把 D.3 形式化为 FATC 控制器，完成可复现消融。

### 任务
1. 固化 FATC 控制器参数接口：g_low, g_high, gamma, conf_thr, conf_slope。
2. 增加两组对照：
   - 无频率自适应（g_low = g_high）
   - 无置信门控（conf 恒为 1）
3. 输出统一指标：PESQ, ESTOI, SDR, 高频 SDR。

### 通过标准
1. 10-15 dB 桶下 delta_sdr_vs_noisy 接近 0（例如大于 -0.5 dB）。
2. 高频 delta_sdr 显著优于并联与 Phase C residual。

## Phase E1: 结构同构（共享主干雏形）
### 目标
构建单主干双头结构的最小可训版本，不再外接串联式 PGUSE 流程。

### 任务
1. Backbone: Transformer + 状态空间分支（可先用轻量替代 Mamba2）。
2. ADD Head: 离散幅度 token 分类头。
3. Flow Head: 连续相位残差速度场回归头。
4. FiLM 交互：Flow 特征调制 ADD 隐层。

### 损失
1. L_add: 离散 token 交叉熵
2. L_flow: 相位残差回归
3. L_mrstft: 复谱重建
4. L_loop: 复域闭环约束

## Phase E2: 任务特异型编解码器 T-CNAC
### 目标
将外部编码器依赖替换为自训复值编解码器，实现潜空间一致性主导。

### 任务
1. 训练复值编码器：幅度离散分支 + 相位连续分支。
2. 建立幅相关联一致性约束。
3. 将 T-CNAC 与 E1 主干端到端连接。

## Phase E3: 全量训练与论文级评测
### 目标
完成可投稿实验矩阵。

### 任务
1. 数据扩容到公开基准及多噪声域。
2. 完成显著性检验、置信区间、效率指标（RTF, Params, VRAM）。
3. 完成消融矩阵：
   - 去掉 FATC
   - 去掉 FiLM
   - 去掉 Flow Head
   - 去掉复域闭环

## 执行建议
1. 先跑 E0 消融，锁定 D.3 最优控制区间。
2. 再做 E1 轻量共享主干，不立即跳到全量大模型。
3. E2 放在有稳定训练资源后推进，避免过早复杂化。
