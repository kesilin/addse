# ADDSE 并联连续适配实验方案与结果（2026-03-26）

## 1. 实验目标

本轮实验的核心目标是：

1. 保持 ADDSE 原有离散扩散主干不变，最大化复用已有预训练能力。
2. 以最小改动加入连续分支并联交互，验证是否能在小样本条件下带来可测的增强收益。
3. 先构建可跑通、可复现、可比较的基础闭环，不做大规模架构重写。

为此，我们采用“离散扩散主干 + 轻量连续门控并联”策略，并在同口径 60 条评估集上与基线对比。

## 2. 对照实验设计

### 2.1 基线模型（Base）

- 模型类型：ADDSERQDiT（原生离散扩散主干）。
- 训练配置来源：已有微调配置。
- 用途：作为并联方案的直接对照组。

### 2.2 实验模型（Parallel）

- 模型类型：ADDSERQDiTParallel。
- 核心思路：在主干前加入连续适配分支，以门控方式校准输入，再进入原离散扩散网络。
- 关键公式：

  x' = x * (1 + alpha * sigmoid(Wg(Adapter(c))))

其中：

1. x 为离散扩散主干输入表征。
2. c 为条件输入表征。
3. Adapter 为 1x1 卷积构成的轻量连续分支。
4. Wg 为门控投影层。
5. alpha 为并联交互强度。

本实验设定：

- interaction_alpha = 0.1
- adapter_hidden = 256

## 3. 数据与样本规模

### 3.1 数据来源与构造方式

1. 采用动态混音数据管线（DynamicMixingDataset）。
2. 语音与噪声分别从流式分块目录采样，再按设定 SNR 合成。
3. 采样率统一为 16 kHz。
4. 语音片段长度统一 1.0 秒。

### 3.2 本轮规模

- 训练集长度：60（用于快速验证）。
- 验证集长度：20。
- 评估样本长度：60。
- SNR 区间：0 dB 到 10 dB。

说明：本轮是“快速可行性验证规模”，用于确认方向，不用于最终论文级统计结论。

## 4. 模型结构说明（完整链路）

### 4.1 离散扩散主干（保持不变）

主干仍是 ADDSE 的 RQDiT 体系，关键特征：

1. 输入投影：将输入与条件映射到隐藏维度。
2. 时间维 DiT：建模序列时域依赖。
3. 码本维 DiT：在多码本维度进行建模（K>1 时启用）。
4. AdaLN 与输出投影：完成条件调制与最终输出映射。

### 4.2 并联连续分支（新增）

新增连续支路结构如下：

1. Adapter：
   - Conv2d(input_channels -> adapter_hidden, kernel=1)
   - SiLU
   - Conv2d(adapter_hidden -> input_channels, kernel=1)
   - SiLU
2. Gate 投影：
   - Conv2d(input_channels -> input_channels, kernel=1)
3. 门控：
   - gate = sigmoid(gate_proj(adapter(c)))
   - x <- x * (1 + alpha * gate)

### 4.3 形状与兼容性

1. 支持 3D 与 4D 输入（内部统一到 4D 处理）。
2. 若输入为 3D，自动加码本维处理后再还原。
3. 主干前向接口与训练框架保持兼容，不改变外部调用方式。

结论：该改造属于“外挂式增强”，对已有预训练与训练脚本侵入很低。

### 4.4 关于 DAC 与连续特征的澄清

本轮实验并未使用 DAC（Descript Audio Codec）作为前端，也不是“先做 DAC 编码再向量化”的路线。

本轮真实链路是：

1. 使用 ADDSE 既有 NAC + RQDiT 离散扩散主干。
2. 离散主干输入仍来自现有 token/embedding 流程。
3. 新增的连续分支来自条件特征 c 的轻量适配（1x1 Conv Adapter），用于门控校准主干输入。

因此，更准确的描述应为：

- 离散语义骨架：由 ADDSE 既有离散扩散主干承载。
- 连续细节校准：由并联 Adapter+Gate 分支提供。

### 4.5 输出语义与重构路径（避免歧义）

1. 在扩散采样过程中，模型直接预测的是离散 token（码本索引）的更新。
2. 连续分支不单独生成一条可听波形，也不走独立解码头。
3. 连续分支仅在潜空间中调制离散主干特征，使离散采样结果包含连续细节引导。
4. 系统最终对外输出仍是重构波形：先得到离散 token，再经冻结 NAC 解码器还原音频。

结论：这是“离散主输出、连续做调制”的非对称并联机制。

## 5. 训练配置与超参数

### 5.1 通用配置

1. 优化器：AdamW。
2. 学习率：1e-4。
3. eps：1e-7。
4. 扩散步数：128。
5. block_size：200。
6. 精度：16-mixed。
7. 梯度裁剪：1.0。
8. EMA：0.999。
9. 设备：单卡 GPU。

### 5.2 本轮并联实验训练策略

1. 从基线微调权重热启动（init-ckpt）。
2. max_epochs = 10。
3. limit_train_batches = 8（快速验证）。
4. limit_val_batches = 2。
5. train batch size = 8，val batch size = 1。

## 6. 实验执行步骤（可复现流程）

### 6.1 准备阶段

1. 新增并联模型类并导出。
2. 新建 60 条数据的并联配置。
3. 指定基线 checkpoint 作为并联初始化权重。

### 6.2 训练阶段

1. 启动并联训练。
2. 若数据流式状态冲突，关闭 resume 并重置随机状态。
3. 训练结束后保留 best 与 last 两类 checkpoint。

### 6.3 评估阶段

1. 以同口径参数分别评估：
   - 并联 best
   - 并联 last
   - 基线 best
   - 基线 last
2. 指标统一：SDR、ESTOI、PESQ。
3. 评估样本统一 60 条，避免比较偏差。

## 7. 运行问题与修复

### 7.1 工具链问题

- 问题：uv 不可用。
- 处理：改用虚拟环境 Python 直接运行。

### 7.2 数据目录索引问题

- 问题：edbase_noise 目录缺少 index.json，训练/评估报错。
- 处理：覆盖噪声目录为 edbase_noise_original。

### 7.3 流式数据恢复状态问题

- 问题：resume 状态触发断言。
- 处理：训练时增加 resume=false 与 reset_rngs=true，配合 overwrite 重新启动。

以上修复后，训练与评估流程均可稳定执行。

## 8. 实验结果（同口径 60 条）

### 8.1 结果汇总

1. 并联 best（epoch=08, val_loss=4.14）
   - ESTOI: 0.62
   - PESQ: 1.55
   - SDR: 0.73

2. 基线 best（epoch=07, val_loss=3.54）
   - ESTOI: 0.63
   - PESQ: 1.51
   - SDR: 0.59

3. 并联 last
   - ESTOI: 0.64
   - PESQ: 1.51
   - SDR: 1.03

4. 基线 last
   - ESTOI: 0.63
   - PESQ: 1.51
   - SDR: 0.48

### 8.2 结果解读

1. SDR：并联方案在 best 与 last 两种 checkpoint 上都显著优于基线，降噪/失真抑制改善明确。
2. PESQ：并联与基线接近，best 情况有小幅提升。
3. ESTOI：整体与基线基本持平，存在轻微波动。
4. checkpoint 选择：并联 last 在本轮指标上优于并联 best，说明 val_loss 最优点不一定对应感知指标最优点。

## 9. 结论

在“仅 60 条训练样本 + 最小架构改动”的约束下，并联连续适配方案已显示出稳定正向收益，尤其体现在 SDR。该结果支持继续沿“离散扩散主干 + 轻量连续并联交互”方向推进。

## 10. 下一步实验计划

### 10.1 参数扫描（优先级最高）

1. interaction_alpha：0.05 / 0.1 / 0.2。
2. 其余配置固定，做同口径 60 条评估。
3. 目标：寻找 SDR 与 ESTOI/PESQ 的 Pareto 平衡点。

### 10.2 冻结策略消融

1. 仅训练 adapter+gate。
2. 待 SDR 稳定后，采用极低学习率（建议 1e-5）仅微调 RQDiT 输出层。
3. 最后再尝试小学习率解冻更大范围参数。
4. 比较稳定性、收敛速度和指标收益。

建议采用两阶段非对称冻结策略：先稳住并联分支与主干对齐，再逐步放开主干，降低早期梯度破坏离散语义建模的风险。

### 10.3 指标体系增强（相位与自然度）

1. 保留 SDR、ESTOI、PESQ 三项主指标。
2. 增加 Phase Distance（PD）或等价相位误差指标，用于判断相位是否被有效修正。
3. 增加 GOMPSNR 指标，用于补充感知自然度层面的判别能力。

说明：当前结果表现为 SDR 强、PESQ 提升有限，新增相位与感知指标可帮助定位瓶颈到底在幅度重建还是相位一致性。

### 10.4 损失函数与样本加权

1. 在原训练损失基础上，加入低权重复数频谱一致性损失（complex spectral consistency）。
2. 对低 SNR（尤其 0 dB 及以下）样本增加权重，提高困难样本鲁棒性。
3. 通过网格搜索控制附加损失权重，避免牺牲 ESTOI。

### 10.5 可信度增强

1. 做按 SNR 分桶统计（低/中/高 SNR）。
2. 导出少量样本做主观听感 A/B（关注金属感、清晰度、残噪）。

## 11. 风险与注意事项

1. 当前样本量小，方差相对较大，结论属于方向性验证。
2. 评估中出现少量 PESQ 与 STOI 的短语音/静音警告，建议后续增加有效语音占比过滤。
3. 建议下一轮至少扩展到 200 条以上评估样本，提升统计稳定性。
